# Quickspaces

A VS Code extension that provides quick access to your workspaces and project spaces, helping you navigate and manage multiple projects efficiently.

## Features

- **Sidebar Activity Panel**: Access your quickspaces directly from the VS Code activity bar
- **Workspace Management**: Organize and quickly switch between your favorite projects and workspaces
- **Quick Access**: Get instant access to your most-used projects without navigating through file dialogs
- **Visual Organization**: View all your spaces in an organized sidebar interface
- **Streamlined Navigation**: Reduce time spent switching between projects and improve workflow efficiency

> Tip: Quickspaces seamlessly integrates into your VS Code workflow, appearing as a dedicated panel in the activity bar for easy access.

## Requirements

- **VS Code**: Version 1.110.0 or higher
- **Operating System**: Windows, macOS, or Linux
- No additional dependencies required - this extension works out of the box!

## Installation

1. Launch VS Code
2. Go to the Extensions view (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for "Quickspaces"
4. Click "Install"
5. The Quickspaces panel will appear in your activity bar

## Usage

Once installed, you'll see a new icon in the VS Code activity bar (left sidebar). Click it to open the Quickspaces panel and manage your project spaces. You can:

- Add new project spaces to quick access
- Click any space to open it in a new or current window
- Organize and customize your spaces for quick navigation

## Extension Settings

This extension contributes the following settings:

* `quickspaces.enable`: Enable/disable the Quickspaces extension
* `quickspaces.showInActivityBar`: Show or hide the Quickspaces icon in the activity bar
* `quickspaces.defaultBehavior`: Set whether spaces open in new or current window (`new` or `current`)
